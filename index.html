O que são tags HTML?

Voltar ao topo

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico,
quebras de linha, objetos, bullet points e muito mais.
Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet.

Tag de h1 à h6

Voltar ao topo

As tags < h1 > a < h6 > são usadas em HTML para definir cabeçalhos ou títulos de diferentes níveis de importância.
Elas variam em tamanho e são usadas para estruturar o conteúdo de uma página da web de forma hierárquica, com < h1 > sendo
o mais importante e < h6 > o menos importante. Aqui está uma visão geral de como cada uma é usada:

  1. h1 é o CABEÇALHO mais importante e normalmente é usado para o título principal da página.

  2. h2 um pouco menos importante que < h1 >, geralmente usado para subtítulos ou seções principais da página.

  3. h3 menos importante que < h2 >, usado para subseções dentro de uma seção < h2 >.

  4. h4 menos importante que < h3 >, usado para subseções dentro de uma seção < h3 >.

  5. h5 menos importante que < h4 >, usado para subseções dentro de uma seção < h4 >.
  6. Menos importante que < h5 >, usado para subseções dentro de uma seção < h5 >.

Tag de parágrafo < p >

Voltar ao topo

A tag < p > é uma das tags mais fundamentais em HTML (HyperText Markup Language) e é usada para definir um parágrafo de texto
em uma página da web. A palavra "p" em < p > significa "parágrafo". Quando você usa a tag < p >, tudo o que está dentro dela é tratado como um parágrafo.
Os navegadores da web aplicam automaticamente uma margem superior e inferior ao texto dentro de uma tag < p >, o que resulta em um espaço claro entre parágrafos adjacentes.

Tag de título < h2 >

Voltar ao topo

O uso dessa tag formatará qualquer texto entre as tags de abertura e de fechamento como um Título 2 (H2),
que é um tipo de subtítulo. Geralmente, um texto utiliza diversos H2, além de H3 e, mais raramente, H4 em diante.

Tag de negrito < b >

Voltar ao topo

Esta tag irá formatar qualquer texto entre as tags como negrito ('b' vem da palavra 'bold').

Tag de itálico < i >

Voltar ao topo

O **elemento HTML < i > **representa uma parte do texto que é destacada do restante por algum motivo, por exemplo,
termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente,
é apresentado com o uso do tipo "itálico".

Tag de sublinhado

Voltar ao topo

A tag "sublinhado" é uma marcação usada em linguagens de marcação como HTML para aplicar um estilo de sublinhado a texto
específico em uma página da web. No HTML, a tag para aplicar sublinhado é < u >. Quando o navegador encontra essa tag,
ele renderiza o texto envolvido com um sublinhado.
